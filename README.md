# PASSWORD GENERATOR COMPLETE GUIDE

## DESCRIPTION
A sleek, modern web application built with a python Flask backend and an advanced Glassmorphism front-end UI. This application securely generates custom random strong passwords 

---

## FEATURES
* **Manual Customization:** Completely manual control over settings; no forced defaults.
* **Flexible Length:** Choose any desired length between 4 and 64 characters.
* **Character Selectors:** Toggle checkboxes manually to include Uppercase, Lowercase, Numbers, or Special Characters.
* **Instant Visibility:** Passwords generate natively as readable plain text, with an option to mask them using a visibility toggle button (`👁`/`🙈`).
* **Interactive Clipboard:** A seamless copy button that instantly captures the password and changes state to "✅ Copied!" for two seconds.
* **Live Strength Assessor:** Dynamic, color-coded visual indicator badges (Red for Weak, Yellow for Medium, Green for Strong) based on strict character complexity metrics.
